Module-1

LAUNCHING INTO CYBER SECURITY MARCH 2022

Module Home

During this module, we will cover the concepts and principles in the field of Cyber Security, the governance and ethical frameworks and competencies required as a Cyber Security professional. It also enables you to engage, experience and envision the current as well as the future development of the discipline. You will not only be equipped with theoretical and practical skills, but this module will also make you aware of the ethical and professional responsibilities as a Cyber Security professional. The module enables you to engage in debate to identify contemporary cyber security-based challenges and the ability to design appropriate solutions.

Unit-1: Introduction to Cyber Security

This week will focus on the drivers of computer-based information namely: types of networks, types of software, and databases; the internet and the World Wide Web and how it works. It discusses the various functions of these components in an information system and the impact of this on the system in the event of a security breach. It begins by examining the considerable shift in paradigm from manual to digital approaches to processing and storing of data, and how this has consequently given rise to cybercrime and the impact of this on organisations and draws on the key concepts of CIA in Cyber Security, which serves as a background to the module.

Unit-2: Cyber Security Issues and Their Implications for Businesses

This unit has the first seminar for the module and will focus on discussing security considerations for a global company. It will involve you applying the knowledge from the week 1 lecturecast to identify potential ethical and regulatory issues an international company will have to deal with and what advice a Cyber Security professional will to offer such a company.

Unit-3: Approaches to Cyber Security Design

This unit focuses on approaches to designing Cyber Security solutions. There are various methods and techniques available to develop security solutions. To be able to design effective solutions, it is important to be able to effectively identify the threats and vulnerabilities which will inform the best solution to adopt. We will be exploring these principles that underpin good security designs.

Unit-4: Threat Identification and Modelling

In this unit, we will be working through exemplar problems where we will apply the knowledge gained in the lecturecast. It will involve working through a case to identify threats and vulnerabilities and apply threat modelling techniques to develop effective security measures to predict, detect, prevent or respond as discussed in unit 3.

Unit-5: Object-oriented Analysis, Design and Programming Techniques for Secure Systems

This week, the focus is on the implementation of security solutions. The lecturecast will introduce object-oriented approaches to secure software design solutions using UML diagrams within the context of Cyber Security. An introduction to programming will also be covered. This includes the application of UML object-oriented modelling techniques, such as use case and sequence diagrams, to develop a design document required for project implementation. The introduction to programming will cover the principles of programming, as well as the implementation of these principles using the Python programming language.
